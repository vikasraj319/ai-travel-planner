🌍 AI Travel Planner
An intelligent AI-powered travel planning application that generates personalized itineraries, interactive route maps, destination insights, restaurant suggestions, and travel recommendations.
________________________________________
✨ Features
•	🤖 AI-generated travel itineraries
•	🗺️ Interactive maps with optimized travel routes
•	📍 Place recommendations and sightseeing spots
•	🍽️ Nearby restaurant discovery
•	🧭 Smart route planning between locations
•	📅 Day-wise itinerary organization
•	🎨 Modern responsive UI
•	⚡ Fast and smooth user experience
•	🌙 Light/Dark theme support
•	🔐 Secure environment variable management

🧠 How It Works
1.	User enters travel destination and preferences.
2.	AI generates a personalized itinerary.
3.	Map service calculates optimized routes.
4.	Nearby attractions and restaurants are displayed.
5.	User explores and customizes the trip.
